2  Terminal : 

python sniffer.py

//only for Linux OS users

Live capture:
sudo python sniffer.py

Live capture on a specific interface:
sudo python sniffer.py --iface eth0

List available interfaces:
python sniffer.py --list-ifaces

Analyze existing PCAP from terminal:
python sniffer.py --pcap packets.pcap
